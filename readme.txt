== Description ==

Ni Youtube video gallery is a plugin for wordpress that allows you to embed youtube video gallery into your website or blog. This is simple plugin for website or blog that use youtube video.

Some Features . 

* Fast And Easy Install
* Automatically Get Youtube ID
* Video And Images From Youtube
* Add, Delete, Update Video
* Add, Delete, Update Category 
* Template Customization.


== Installation ==

1. Use the WordPress plugin installer or upload files to WordPress plugin directory.
1. Activate plugin from admin panel.
1. Copy youtube video URL 
1. Add New Video, Select Category.
1. Create new Page and Select Template.
1. Add Page to menu.
1. Display video on front end.
